# AniPlayApp
Um stream de animes totalmente gratuito sem anuncios e sem planos pagos. Com vários animes disponiveis desde os mais recentes aos mais populares, disponiveis em sua grande maioria em portugues dublado, os que não forem dublados ainda sim são legendados. Esperamos que goste e apoie esse projeto!

## Requisitos
- Windows 11/10
- Interpretador Python 3+
- NodeJS

## Como usar
1. Clone o repositorio usando ``git clone https://github.com/TheLuke013/AniPlayApp.git``;
2. Abra a pasta do repositorio, e na raiz execute o arquivo ``install-dependencies.bat`` para instalar todas dependencia necessarias;
3. Em seguida, execute o arquivo ``setup-api.bat`` para fazer build da api de animes;
4. Por fim, na raiz do projeto, vá até a pasta ``app`` e execute o arquivo ``run.bat`` para iniciar a aplicação.
